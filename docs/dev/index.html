Preact Hint

Preact-Hint is a lightweight and extensible tooltip component for preact. It was designed with a focus on size, adaptability, and ease of use.

Installation

$ yarn add preact-hint

General Usage

Options

• attribute

Default: 'data-hint'

• template

Default: undefined
